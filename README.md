# hello-world-http-node

Simple project using built-in Node.js `http` module.

# How to start

1. Clone this repo:

        git clone https://github.com/dkostmii/hello-world-http-node.git && cd hello-world-http-node

2. Install dependencies:

        yarn

    or

        npm install

3. Run server:

        npm run start

    or

        yarn start
